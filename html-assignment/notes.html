< p> < / p>

< a > href=""> < / a >
This is an Attribute that contains a hyper link to another page
< p > < / p >

< blockquote > < / blockquote >
This is a Blockquote, used for content quoted from another source
< aside > < / aside >
This is an Aside tag, this is a section for content that is not related to the main topic
